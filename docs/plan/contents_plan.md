# HTML、CSS、JSを理解するための学習教材紹介Webサイト


## 概論

- 全体像を把握する
- それぞれの関係を理解する
- HTMLを理解する
- CSSを理解する
- JaVaScriptを理解する
- [補足]フロントエンドとバックエンド

## 演習

1. 企画を考える
2. 構成をざっくり考える
3. ページ構成を考える
4. 各ページの掲載内容を考える
5. 各ページの内容を構造化する
6. ページのヘッダー、フッター、ナビゲーション、タイトル、内容などのパーツを考える
7. 5.と6.を使いHTMLファイルを作る
8. ページのヘッダー、フッター、ナビゲーション、タイトル、内容などのパーツのデザインを考える
9. 8．を元にして、HTMLを編集しながらCSSを作る
